# Dasbor Baterai NASA
Dasbor ini dibangun untuk menampilkan data spesimen baterai Lithium-ion yang dikenakan percobaan pengosongan dan pengisian muatan oleh _Prognostics Center od Excellence_ (PCoE), instansi di bawah _National Aeronautics and Space Administration_ (NASA). Aplikasi dasbor ini dibangun dengan menggunakan Python dan pustaka utama Streamlit serta bantuan dari Devin AI.
